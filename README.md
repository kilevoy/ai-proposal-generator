# 📄 AI-генератор коммерческих предложений

> Данные об объекте и позициях → готовое коммерческое предложение за секунды: продающий оффер (AI), расчёт с НДС и условиями, в современном интерактивном веб-формате и PDF.

```
📋 Данные → ✍️ AI-копирайтинг → 🧮 Расчёт (НДС, скидка, оплата) → 📄 КП (веб + PDF)
```

Подготовка КП вручную в Word занимает часы, документы выходят разнородными, а отвечать клиенту нужно быстро (скорость ответа напрямую влияет на win-rate). Инструмент собирает профессиональное КП мгновенно и единообразно.

---

## С учётом трендов 2026

- **Скорость** — КП за секунды (быстрый ответ = выше конверсия).
- **AI-копирайтинг** — продающий оффер с выгодой, блок «о компании» и преимущества под конкретного клиента (LLM).
- **Интерактивный веб-формат** — КП как веб-страница с кнопкой «Принять» (подход Qwilr/PandaDoc), а не статичный файл.
- **Аналитика просмотра** — концепт индикатора «клиент открыл КП».
- **Корректный расчёт** — сумма, скидка, НДС (2026 — 22%), график оплаты «аванс + остаток».

## Что делает

- **AI-тексты** — оффер, «о компании», преимущества генерируются под входные данные.
- **Расчёт** — чистые функции (`compose`): суммы, скидка, НДС, график оплаты. Покрыты тестами.
- **Рендер** — интерактивная HTML-страница КП (`render`), готовая к печати в PDF.

## Установка и запуск

```bash
pip install -r requirements.txt
cp .env.example .env        # ключ LLM
```

```bash
# сгенерировать КП (AI заполнит тексты, если их нет в JSON):
python -m kp sample/example.json --number КП-042

# без AI (тексты берутся из JSON):
python -m kp sample/example_filled.json --no-ai
```

Результат — `out/<имя>.html`: откройте в браузере → «Печать» → сохранить в PDF.

## Архитектура

| Модуль | Роль |
|--------|------|
| `kp/model.py` | Модель данных (КП, позиции) |
| `kp/compose.py` | Расчёт: суммы, скидка, НДС, график оплаты |
| `kp/ai.py` | AI-генерация оффера / «о компании» / преимуществ |
| `kp/render.py` | Рендер интерактивной HTML-страницы КП |
| `kp/__main__.py` | CLI |
| `web/` | Веб-демо (React + TypeScript) |

## Тесты

```bash
pip install pytest
pytest tests/ -q
```

Покрывают расчёт (суммы, скидка, НДС, график оплаты, формат денег), модель и рендер. Запускаются в CI.
